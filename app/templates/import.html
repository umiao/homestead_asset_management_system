Your file should have the following columns (tab-separated for TSV, comma-separated for CSV):

  • name - Item name (required)
  • category - Category (e.g., Food, Tools, Cleaning)
  • quantity - Quantity amount
  • unit - Unit of measurement (count, kg, g, liter, ml, etc.)
  • location_path - Hierarchical path (e.g., "Kitchen > Fridge > Top Shelf")
  • acquired_date - Date acquired (YYYY-MM-DD format)
  • expiry_date - Expiration date (YYYY-MM-DD format)
  • notes - Additional notes

Quick Tips

  • Use hierarchical paths with " > " separator (e.g., "Kitchen > Fridge > Door")
  • Dates should be in YYYY-MM-DD format (e.g., 2025-01-15)
  • Leave expiry_date empty for non-perishable items
  • The system will automatically create locations if they don't exist
  • Duplicate items will be created as separate entries (merging coming soon)
